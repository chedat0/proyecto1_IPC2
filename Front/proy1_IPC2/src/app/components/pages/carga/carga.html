Carga de Datos

Importar datos iniciales desde archivo de texto

Formato del Archivo

El archivo debe ser .txt con codificación UTF-8. ℹ️ Cada línea es una instrucción. Las líneas que empiezan con # son comentarios.
USUARIO("username","password",rol) rol: 1=Cliente, 2=Operaciones, 3=Admin
DESTINO("nombre","pais","descripcion")
PROVEEDOR("nombre",tipo,"pais") tipo: 1=Aerolínea, 2=Hotel, 3=Tour, 4=Traslado, 5=Otro
PAQUETE("nombre","destino",dias,precio,capacidad)
SERVICIO_PAQUETE("paquete","proveedor","descripcion",costo)
CLIENTE("dpi","nombre","dd/mm/yyyy","tel","email","nacionalidad")
RESERVACION("paquete","agente","dd/mm/yyyy","dpi1|dpi2")
PAGO("RES-00001",monto,metodo,"dd/mm/yyyy") metodo: 1=Efectivo, 2=Tarjeta, 3=Transferencia
⚠️ El orden importa: USUARIO → DESTINO → PROVEEDOR → PAQUETE → SERVICIO_PAQUETE → CLIENTE → RESERVACION → PAGO
